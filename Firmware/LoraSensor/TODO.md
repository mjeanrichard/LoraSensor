## Critical

- [ ] **Fix NULL dereference in `processData` when JSON parse fails**
  `LoraClient.cpp:161` ‚Äî `cJSON_Parse` can return NULL; execution continues and passes NULL to `cJSON_GetObjectItem`, causing a crash. Add `return ESP_FAIL;` after the `ESP_LOGE` on line 163.

- [ ] **Fix wrong `CMD_SOFT_RESET` constant in SHTC3**
  `SHTC3.cpp:8` ‚Äî `CMD_SOFT_RESET` is `0xEFC8`, same as `CMD_READ_ID`. Correct value per datasheet is `0x805D`. As-is, `reset()` reads the chip ID again instead of resetting the sensor.
